Discord Bots?

Discord is a browser and desktop application designed for instant communication. Utilizing some programming, I started using it to annoy my friends using the built-in bot system. Bots are typically designed by developers and their purpose is to provide utility and fun to group chats called "servers". Just because a bot is meant to be useful, does not mean you are forced to make it useful. When I was in middle school, there used to be a joke when someone said, "What?". It became a race to see who could reply with "Chicken butt." the fastest. By utilizing the power of a Node.js library, and my need to pester my friends, I created my first-ever Discord bot that's only purpose was to do this.

What's Your Story?

I originally started making bots because I was extremely bored one night, probably messing around with some bots that my friend had added. After mindlessly typing command after command because of how enamored I was of the little game someone made with a bot, I realized that it was probably possible to do it myself. I had already been studying how to code for a small period of time now, and I figured there wasn't a better way to not only show off to my friends but also get practice in as well. So I sat in my bed, looked up how to make Discord bots, and the rest is history. I made my first ever bot with "discord.js" which is a Node.js library for Discord bots. I vividly remember rolling in my bed in excitement watching as my little robot came to life and said little messages.
My first bot was horribly unoptimized, as everytime a message was sent it would run atleast 23 "if statements" which use a lot of resources very quickly. One if statement for every little command.

How Do I Make A Discord Bot?

In almost every bot I've ever made beyond my first bot, I've never touched discord.js again. I decided to switch to a library for python, being "discord.py". To create a bot you'll need the following,

  • An account on replit.com(or anything that can host a website and can run code)
  • An account on Discord

To start, head to this website, and in the top right lies a "New Application" button. When you click it the following popup will appear:

The next step beyond this is to go to the Bot ribbon and then create a bot.

At this point you can customize your bot's profile to your liking, but for the sake of this tutorial I will be leaving it as the default. It is now time for the hard part, the code. On your bot page there should be an option to reset the bots "token". Reset it, copy the string of text it gives you, and bury that string in the deepest depths of your computers harddrive. It is essentially a password to your bot, and once it's out, your bot is going to be vulnerable to any average Joe



Now go to the replit website and press on the button that says "+Create". Choose the python option and name it whatever you like.

You should be faced with the code editor. On the left side there is a lock symbol, click on it
